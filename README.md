# Nigeria Tax Law 2026 Calculator & Guide

A simple, user-friendly web app to help Nigerians understand and calculate their taxes under the new 2026 Tax Law reforms.

**Live Demo:** [https://dayaki.github.io/new-tax-helper/](https://dayaki.github.io/new-tax-helper/)

## Features

- **Tax Calculator** — Enter your annual income and see exactly what you owe under the new law
- **Old vs New Comparison** — See how much more you keep compared to the old tax system
- **Deductions Support** — Factor in pension, NHF, and rent relief
- **Layman's Guide** — Plain-English explanation of the 2026 tax bands and reliefs
- **FAQs** — Answers to common questions about the new tax system
- **Resources** — Official links and step-by-step guide to paying your taxes

## Tax Bands (2026)

| Income Band       | Tax Rate      |
| ----------------- | ------------- |
| First ₦800,000    | 0% (Tax Free) |
| Next ₦2,200,000   | 15%           |
| Next ₦9,000,000   | 18%           |
| Next ₦13,000,000  | 21%           |
| Next ₦25,000,000  | 23%           |
| Above ₦50,000,000 | 25%           |

## Tech Stack

- **Framework:** Next.js 16 (App Router)
- **Styling:** TailwindCSS 4
- **Icons:** Lucide React
- **Language:** TypeScript

## Getting Started

```bash
# Install dependencies
pnpm install

# Run development server
pnpm dev

# Build for production
pnpm build
```

Open [http://localhost:3000](http://localhost:3000) to view the app.

## Deployment

This project auto-deploys to GitHub Pages via GitHub Actions on every push to `main`.
